# Mobile App — BahBar (Expo / React Native)

Application mobile Expo. Consomme l’API BahBar via `EXPO_PUBLIC_BASE_URL`.


## Prérequis
- Node.js 18+ et npm


## Installation
```bash
cd mobile-app
npm install
```

## Variables d’environnement
Exemple .env :
EXPO_PUBLIC_BASE_URL='http://172.20.10.5:5001/api'

## Démarrer l’application
```bash
npm run start 
```

## Tests
```bash
npm run test
```
